# Homebrew Tap

Homebrew tap for applications by [walcew](https://github.com/walcew).

## Installation

```bash
brew tap walcew/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `openfortivpn-connect` | macOS GUI client for openfortivpn VPN connections |

### Install a cask

```bash
brew install --cask walcew/tap/openfortivpn-connect
```
